# Dynamo_Sample_NodeJS
This is simple nodejs sample for DynamoDB using aws sdk.


## Usage

```:init
npm i
```

```launch dynamodb at local machine
docker-compose up -d
```

Edit index.js to use following methods
- queryData
- getData
- putData
- loadData
- updateData
- createTable
- deleteTable


```lauch script
node index.js
```
